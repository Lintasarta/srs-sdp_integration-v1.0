# Software Interface

For this Service Desk Portal Integration:

- **Frontend:** The Cloudeka Service Portal (ai.cloudeka.id) will serve as the user-facing interface for ticket submission and monitoring.
- **Backend (Ticketing System):** ManageEngine ServiceDesk Plus (SDP) will be the core of the operation team's ticketing system. This system will be responsible for:
  - Storing and managing all ticket details, status, and related data.
  - Handling ticket creation, updates, and closure.
  - Managing discussion entries and attachments.
  - Implementing authentication and authorization for operation team access.
- **Integration Platform:** The integration will leverage standard web technologies and APIs.
